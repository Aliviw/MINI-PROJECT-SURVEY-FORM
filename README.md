 Student Language Survey Form

A simple web-based form designed for academic institutions to collect student language preferences during subject selection.

📌 About the Project

This is my first web development project, created to practice basic front-end skills using:

HTML

CSS

Minor  JavaScript 

The form simulates a real-world scenario where a school gathers information from students about language choices.

🎯 Features

📝 Clean and simple student survey form

🎨 Styled using CSS for better user experience

📱 Basic responsive layout

⚡ Beginner-friendly structure and code

🛠️ Technologies Used

HTML5 – Structure of the form

CSS3 – Styling and layout

JavaScript – To enable saving and processing of form data

🚀 Future Improvements

Connect to a backend or database

Improve responsiveness for all devices

Enhance UI/UX design

📂 Project Purpose

This project was built as a practice exercise to:

Understand form creation in HTML

Apply CSS styling techniques

Prepare for more advanced web development projects
